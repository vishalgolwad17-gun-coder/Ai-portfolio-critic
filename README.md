## 📖 About

AI Portfolio Critic is a web application that helps developers evaluate and improve their portfolio websites using Artificial Intelligence.

Users can submit a portfolio URL along with a screenshot of their website. The AI analyzes the portfolio and generates a comprehensive report with scores, strengths, weaknesses, and personalized recommendations.

The goal is to help developers create better portfolios that leave a strong impression on recruiters and hiring managers.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/portfolio-critic-ai.git
```

Navigate to the project

```bash
cd portfolio-critic-ai
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

## 🧠 How It Works

1. Enter your portfolio website URL.
2. Upload a screenshot of your portfolio.
3. AI analyzes the website.
4. Generates detailed scores and feedback.
5. View your report.
6. Export the report as a PDF.
